# Patient Observer

Estimates posture on the bed using Kinect.

## Timeline

- **2017** — Presented at the 2017 JSPE Spring Meeting.
- **2016** — Started as graduation research.

## Prerequisites

- This project is **Windows-only**.
- Besides Kinect for Windows v2, [Visual Studio](https://visualstudio.microsoft.com/) and [Kinect for Windows SDK 2.0](https://www.microsoft.com/en-us/download/details.aspx?id=44561) are required.

## Reference

長島 弘昂, 藤尾 三紀夫, 梅本 琢也, 患者の動向監視支援システムの開発, 精密工学会学術講演会講演論文集, 2017, 2017S 巻, 2017年度精密工学会春季大会, セッションID C77, p. 227-228, 公開日 2017/09/01, https://doi.org/10.11522/pscjspe.2017S.0_227, https://www.jstage.jst.go.jp/article/pscjspe/2017S/0/2017S_227/_article/-char/ja, 抄録: 近年高齢化が進み，認知症患者の不用意な行動による事故が多発している．対策として現在は離床センサが利用されているが，検知が離床後になることや誤作動が多いことから正確に患者の状態を把握できていない．そこで本研究では，連続的に患者の状態を取得することで患者の状態監視を支援するシステムの開発を目的としている．本報では距離センサを用いて患者の5状態をリアルタイムで識別する基本システムを構築したので報告する．
